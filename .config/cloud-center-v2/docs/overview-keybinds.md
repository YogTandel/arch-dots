# Overview Keybinds

| Key | Action |
|-----|--------|
| `Super+Tab` | Toggle overview |
| `← → / h l` | Navigate workspaces (left/right) |
| `↑ ↓ / k j` | Navigate rows (up/down) |
| `1–9, 0` | Jump to workspace in current group (0 = 10th) |
| `Scroll` | Cycle workspaces |
| `Esc / Enter` | Close overview |
| `Tab / Shift+Tab` | Cycle through open windows by focus history |
| Click workspace | Switch to that workspace |
| Click window | Focus window and close overview |
| Middle-click window | Close window |
| Drag window | Move window to another workspace |
